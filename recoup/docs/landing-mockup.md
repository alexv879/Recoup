- Carousel padding: 40px vertical, 20px horizontal

## A/B TESTING RECOMMENDATIONS

**Test 1: Carousel vs. Static Grid**
- Option A: Auto-rotating carousel (current)
- Option B: Static 3-card grid (no rotation)
- Measure: Time on page, clicks, scroll depth

**Test 2: Short Quote vs. Full Story**
- Option A: 1-sentence quote (current)
- Option B: 2-3 sentence full quote
- Measure: Read time, engagement, conversion

**Test 3: Number of Testimonials**
- Option A: 3 rotating cards
- Option B: 5 rotating cards
- Measure: Scroll depth, carousel interaction rate

**Test 4: Social Proof Metric Order**
- Option A: 8,500+ Users | Â£2.3M Recovered | 3-Day Resolution
