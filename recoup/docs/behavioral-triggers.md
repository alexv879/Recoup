## BEST PRACTICES FOR BEHAVIORAL EMAILS

1. **Personalization:** Always include [Client Name], [Amount], or [Action User Took][57]
2. **Specificity:** Reference exact behavior: "You created an invoice" vs. "Your account is ready"[57]
3. **Urgency without pressure:** Use "48 hours left" not "DO THIS NOW"[53]
4. **Single CTA:** One primary action per email; secondary CTA is optional[35]
5. **Tone:** Stay friendly, helpful, never pushy[54]
6. **Frequency cap:** Don't send more than 1 behavioral trigger per user per day[53]
7. **Segmentation:** Use user metadata (plan, activity level) to segment triggers[35]
