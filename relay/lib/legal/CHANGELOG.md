# Legal Documents Changelog

This file tracks all changes to Recoup's legal documents for transparency and compliance.

## Version Control Policy

- **Version Format**: `MAJOR.MINOR` (e.g., 1.0, 1.1, 2.0)
- **MAJOR**: Significant changes requiring user re-acceptance
- **MINOR**: Clarifications, corrections, or non-material changes
- **Review Cycle**: Annual review (November each year)
- **Notification**: 30 days advance notice for material changes via email

## UK GDPR Compliance

Under UK GDPR Article 12, we must provide:
- Concise, transparent, intelligible information
- Easily accessible privacy information
- Clear indication of any changes to policies

## Notification Policy

### Material Changes
When we make material changes to legal documents, we will:

1. **Advance Notice**: 30 days minimum via email to all active users
2. **In-App Notification**: Prominent notice on dashboard
3. **Acceptance Required**: For Terms of Service changes, may require re-acceptance
4. **Changelog**: Update this file with detailed change description
5. **Version Archive**: Previous versions available on request

### Non-Material Changes
For clarifications, typos, or formatting:

1. **No Advance Notice Required**: Changes take effect immediately
2. **Changelog Updated**: Document all changes in this file
3. **Version Number**: Increment MINOR version only
